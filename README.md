# 🌤️ Weather App

A modern Flutter weather application that provides real-time updates using the **OpenWeather API**.  
The app shows live location-based weather, forecasts, and alerts with a clean and gradient UI.

---

## 🚀 Features

- 📍 **Home Screen** – Live weather updates using your current location
- 🔍 **Search** – Find weather by city or country (centered modern search bar)
- 📅 **7-Day Forecast** – View detailed upcoming weather trends
- ⚠️ **Alerts** – Notifications for extreme weather conditions
- 🛰️ **Radar View** – Real-time cloud and rain maps
- 🎨 **Settings** – Theme (Light/Dark) and unit (°C/°F) customization

---

## 🛠️ Tech Stack

- **Flutter** (Dart) – UI & app development
- **OpenWeather API** – Weather data source
- **Geolocator** – Detect user’s live location
- **HTTP** – Fetch weather data from API
